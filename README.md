# SkillCraft Technology - Task 2: Data Cleaning and Preparation

## Objective
Perform data cleaning and preparation on the Global Superstore dataset using Python and Pandas.

## Tasks Performed
- Loaded dataset directly from GitHub into Google Colab.
- Performed data inspection using:
  - df.head()
  - df.info()
  - df.describe()
- Handled missing values using dropna().
- Removed duplicate rows using drop_duplicates().
- Converted Order Date from string to datetime format.
- Exported cleaned dataset to a new CSV file.

## Tools Used
- Python
- Pandas
- Google Colab
- GitHub
